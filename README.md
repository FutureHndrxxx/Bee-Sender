# ERC-20 Airdropper

ERC-ERC20 Token Multi-Chain Airdropper. Sends tokens to multiple wallets in one transaction to save gas.

Support networks:<br>
Polygon, Huobi, Fantom, BSC mainnets and Ethereum Ropsten testnet.
<p>
Max addresses to airdrop at once:<br>
300-500
